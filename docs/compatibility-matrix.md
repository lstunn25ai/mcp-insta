# Матрица возможностей

| Возможность | Условие | Статус |
| --- | --- | --- |
| Профиль и медиа | `instagram_basic` и успешная диагностика | Поддерживается |
| Аналитика | `instagram_manage_insights` и успешная диагностика | Поддерживается |
| Чтение Direct | Page access token и Instagram Messaging API | Поддерживается после OAuth |
| Ответ в Direct | Сначала `ig_direct_reply_prepare`, затем отдельный `ig_direct_reply_confirm` | Поддерживается |
| Чтение комментариев | Отдельная реализация и проверка разрешений | Недоступно |
| Публикация, Threads, webhooks | Не входят в проект | Недоступно |

Meta может ограничивать Direct только диалогами, которые пользователь начал сам. Перед использованием Direct подтвердите права приложения и доступ Page к Instagram аккаунту.

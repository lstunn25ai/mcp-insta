# Чек-лист первого публичного релиза

## Перед commit

- [ ] `npm ci` выполнен по lockfile.
- [ ] `npm run check` проходит.
- [ ] `npm audit --omit=dev --json` не содержит уязвимостей.
- [ ] `npm pack --dry-run --json` не включает исходники отключённых функций, секреты или логи.
- [ ] `git diff --check` проходит.
- [ ] README и публичные документы имеют UTF-8 без BOM и корректно отображаются на GitHub.
- [ ] Проверены `.gitignore`, `git status`, `git diff` и `git ls-files`.

## Перед push

- [ ] Remote указывает на независимый репозиторий владельца, а не на чужой проект.
- [ ] Реальные `.env`, Credential Manager данные, OAuth callback URL с параметрами и runtime-логи не попадают в commit.
- [ ] Создан отдельный commit с проверенным списком файлов.
- [ ] После push проверен отрендеренный README на GitHub.

## Проверка live-интеграции

- [ ] Владелец завершил OAuth и `insta_diagnose`.
- [ ] Профиль, медиа и доступная аналитика проверены read-only запросами.
- [ ] Direct проверен чтением диалогов без отправки сообщения.
- [ ] Любой ответ Direct использует `prepare`, просмотр результата и отдельный явный `confirm`.

# Security and release notes

Дата проверки: 2026-07-19.

## Границы безопасности

- Секреты и токены хранятся только в Windows Credential Manager.
- SQLite не принимает поля с токенами, cookie, URL, паролями и текстом сообщений.
- OAuth использует PKCE, одноразовый loopback callback и ограниченный срок ожидания.
- Ошибки и MCP-ответы редактируются, чтобы не раскрывать токены и параметры URL.
- Direct читает только ранее перечисленные объекты. Отправка возможна лишь через `prepare` → отдельный `confirm`.

## Публичная поставка

- В исходном дереве нет устаревших команд публикации, Threads, webhook или конфигураций с токенами в environment variables.
- Пакет содержит только собранный runtime, документацию, лицензию и миграции, необходимые для работы.
- Перед публикацией должны быть выполнены `npm run check`, `npm pack --dry-run --json`, проверка Git diff и сканирование tracked-файлов на секреты.

## Зависимости

Перед каждым релизом зависимости проверяются отдельно: обновление выполняется только после совместимого обновления lockfile и повторного полного тестового прогона.
